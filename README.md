## Syllabus
**Parte I: Introduzione**
- Concetti di base
- Storia dei sistemi operativi
- Relazione tra macchina fisica (HW) e sistemi operativi

- CPU Scheduling
- Threads

- Implementazione del File System
- Struttura del File System
